## 🎯 專案目標

在規劃登山或步道行程時，使用者需要同時查詢：

- 🏞 步道資訊（難度、位置、路線）
- 🌦 即時天氣（是否下雨、溫度）
- 📍 是否適合出發

👉 這些資訊通常分散在不同網站

### ✨ 本專案解決的問題
將「步道 + 天氣」整合在同一個畫面，降低決策成本。

## ⚙️ 主要功能

### 🏞 步道探索
- 串接台灣步道資料 API
- 支援篩選與列表瀏覽
- 卡片式資訊呈現

### 🌦 即時天氣整合
- OpenWeatherMap API 串接
- 每條步道顯示對應天氣資訊

### 🎯 決策輔助 UI
- 步道 + 天氣整合顯示
- 降低查詢切換成本

### 📱 響應式設計
- 支援手機 / 平板 / 桌機
